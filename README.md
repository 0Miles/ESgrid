ES grid
===
ES不是easy，是eighty seven

這東西我自己給87分，不用再高了。

## Format
![grid](http://i.imgur.com/6nYvMD5.jpg)  
  
Column Width: 6%  
Gutter Width: 2%  
Grid Padding: 3%  

## Installation  

在你的sass文件裡import它  
```sass
@import ESgrid
```

## Example
```html
<div class="grid">
    <div class="col xs-12 gutters-sm-12 gutters-md-6 gutters-lg-3"></div>
    <div class="col xs-12 gutters-sm-12 gutters-md-6 gutters-lg-3"></div>
    <div class="col xs-12 gutters-sm-12 gutters-md-6 gutters-lg-3"></div>
    <div class="col xs-12 gutters-sm-12 gutters-md-6 gutters-lg-3"></div>
</div>
```
